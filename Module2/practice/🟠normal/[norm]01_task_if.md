## "Трехзначное ли"

### Задание

Проверьте, является ли данное число трехзначным.

### Формат входных данных

Дано целое число.

### Формат выходных данных

Вывести "Да", если данное число трехзначное и "Нет" в противоположном случае.

### Решение задачи

```python
num = int(input("Введите целое число: "))

if num >= 100:
    print("Да")
else:
    print("Нет")
```
